My Projects

Number Guessing Game

A simple and interactive Java-based number guessing game where the player tries to guess a randomly generated number between 1 and 100. The game allows up to 5 attempts and provides feedback after each guess, indicating whether the guessed number is less than or greater than the target number. If the player guesses correctly within the allowed attempts, they win; otherwise, the correct number is revealed after the attempts are exhausted. This project demonstrates basic programming concepts, engaging way to practice and improve problem-solving skills.

Online Reservation System

This Java-based console application enables users to efficiently manage seat reservations. It features a user-friendly menu for viewing the seat map, reserving seats, and canceling existing reservations. The application handles seat availability and provides feedback for invalid or duplicate actions. Key functionalities include displaying reserved and available seats, ensuring seat number validation, and confirming reservations or cancellations. This project demonstrates essential Java programming concepts, offering a practical example of building interactive applications.
